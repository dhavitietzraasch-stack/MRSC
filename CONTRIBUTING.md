# Como Contribuir

O MRSC é um projeto comunitário. Todas as contribuições são bem-vindas!

## Sugerir Novas Sintaxes

1. **Crie um exemplo prático** mostrando o problema atual
2. **Proponha a nova sintaxe** com 3-5 casos de uso
3. **Explique o impacto** em regras existentes
4. **Abra uma Discussão** no GitHub

## Reportar Ambiguidade

```
Bug: [descreva a ambiguidade com exemplo]
Regra atual: [exemplo que falha]
Comportamento esperado: [o que deveria acontecer]
```

## Submeter Traduções

- Arquivos em `prompts/`
- Mantenha **exata** fidelidade técnica
- Teste colando na IA antes de submeter

## Regras para Pull Requests

✅ **Obrigatório:**
- Incluir **exemplo funcional**
- **Justificativa clara** (por que essa mudança?)
- **Impacto em versões anteriores** (breaking changes?)
- Testado em pelo menos 2 IAs diferentes

❌ **Evite:**
- Mudanças que quebrem sintaxe existente
- Sintaxes muito complexas (mantenha leve)
- Sugestões sem exemplo prático

## Teste Antes de Submeter

1. Cole o prompt na IA
2. Declare 5-10 regras variadas
3. Teste todos os cenários do seu PR
4. Confirme que regras antigas continuam funcionando

**Obrigado por tornar o MRSC melhor! 🚀**
